# Color Stock

カラーパレット管理ツール

## 技術スタック

- Next.js (App Router)
- TailwindCSS
- shadcn/ui
- Framer Motion

## 開発環境のセットアップ

```bash
# リポジトリのクローン
git clone <repository-url>
cd color-stock

# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

## 機能

- カラーパレットの作成と保存
- カラーコードのコピー
- パレットの管理
